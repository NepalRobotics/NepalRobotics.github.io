Nepal Robotics

Bullis-Kanjirowa STEM Project

About Us

We are a consortium of students, academics, and industry leaders working largely on a voluntary basis to develop Unmanned Aerial Systems as tools for the public good. What started out as an educational project has begun to develop into something more http://nasawatch.com/archives/2015/03/nasa-family-pur.html.

Our Goals

This project runs along two tracks: first, we wish to experiment with UAS as a platform to aid in search and rescue efforts and, second, we wish to increase engagement with and collaboration about UAS in institutes of learning around the world. To these ends, students at the Rensselaer Polytechnic Institute will be developing an open source project to create autonomous vehicles capable of aiding rescuers in the search for buried avalanche victims in the Everest region of Nepal.

Project Information

You can find some more information about the main project at http://nepalrobotics.org/. You can also find the RPI-specific project information at our GitHub documentation wiki: https://github.com/NepalRobotics/ProjectDocumentation/wiki. RPI's contribution to the Nepal Robotics Project will be largely independent of the other, ongoing, projects which are more focused on STEM education. Our project at RPI will be focused on developing a practical implementation.

Project Implementation

All software written as part of the RPI project will be made available under an open MIT license on GitHub: https://github.com/NepalRobotics.

The goal is to develop a software and hardware package that integrates a Software Defined Radio direction finding array with a Raspberry Pi and a 3D Robotics Aero UAV.

The core software stack will be written in Python 2.x, and targeted to the open-source DroneKit library.
